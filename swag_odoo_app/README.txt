🚀 SWAG ODOO APP - STREAMLIT REPLICA

INSTALL:
1. Extract swag_odoo_app.zip → custom_addons/
2. Restart Odoo
3. Apps → "SWAG Product Comparison" → Install

✅ 4 Odoo sync (SWAG/La Rouche/DiffClothes/FashionLimits)
✅ Product search + filters
✅ PDF invoice parsing
✅ Excel export
✅ Dark theme + Arabic

CONFIG:
Settings → SWAG Dashboard → Config → Add 4 URLs
